Star Battle basics

Think of it as a tiny placement puzzle: the whole board needs 7 stars, but the rows, columns, color areas, and neighbors all restrict where they can go.

  • Place exactly one star in each row.
  • Place exactly one star in each column.
  • Place exactly one star in each color region.
  • Stars may not touch, including diagonally.

FAQ

Common Questions

What is the goal of Star Battle?

Place one star in each row, one star in each column, and one star in each color region. Stars cannot touch each other, including diagonally.

Can a generated puzzle have no solution?

The generator checks every board with a solver before showing it. A puzzle is accepted only when the solver finds exactly one valid solution.

How do I use the Killer Sudoku cage calculator?

Type the target sum, choose how many cells are in the cage, and keep only the digits that are still available. The calculator lists all matching combinations.

Do I need an account?

No. The daily puzzle, unlimited practice puzzles, hints, checks, sharing, and calculator all work in the browser without an account.
