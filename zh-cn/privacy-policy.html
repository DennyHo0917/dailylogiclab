隐私政策

最后更新： June 19, 2026

Daily Logic Lab 提供免费的浏览器逻辑谜题和解题辅助工具。本政策说明你使用网站时可能涉及的信息。

你主动提供的信息

Daily Logic Lab 当前不要求注册、登录或付款。如果你通过 GitHub Issues 联系网站，你发布的信息由 GitHub 处理，并可能是公开的。

浏览器本地存储

谜题体验可能在你的浏览器中保存少量数据，例如连胜、最佳时间、已完成日期、提示次数和近期浏览器内事件。你可以通过清除浏览器站点数据来删除。

分析统计

Daily Logic Lab 使用 Google Analytics 了解聚合使用情况，例如页面浏览、开始题目、使用提示、检查盘面以及一般设备或浏览器信息。

广告

Daily Logic Lab 使用 Google AdSense 展示广告。Google 及其合作伙伴可能使用 Cookie 来投放、衡量和个性化广告。你可以通过 Google 广告设置查看和管理广告个性化。

第三方链接

网站可能链接到 GitHub、Google 文档或其他外部页面。Daily Logic Lab 不负责外部网站的隐私做法。

儿童隐私

Daily Logic Lab 是面向一般用户的谜题网站，不会有意收集儿童个人信息。
